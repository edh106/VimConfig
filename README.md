# VimConfig
My python VIM config

Instructions

install vim if not already installed

Grab Vundle

`git clone https://github.com/gmarik/Vundle.vim.git ~/.vim/bundle/Vundle.vim`

Grab config

`git clone https://github.com/edh106/VimConfig.git`

Run Vim to install plugins

:PluginInstall
